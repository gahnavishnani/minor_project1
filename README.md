# Laptop Price Prediction - Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Laptop Price Prediction dataset. The objective is to analyze the dataset, clean the data, visualize important patterns, and prepare the data for future machine learning tasks.

## Dataset
- **Dataset Name:** Laptop Price Prediction Dataset
- **Source:** https://www.kaggle.com/datasets/jacksondivakarr/laptop-price-prediction-dataset

## Tools & Libraries Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## EDA Steps Performed
- Imported required libraries
- Loaded the dataset
- Structural inspection (`head()`, `info()`, `describe()`, `shape`, `columns`)
- Checked for missing values
- Checked for duplicate records
- Removed unnecessary columns
- Performed univariate analysis
  - Histogram
  - Box Plot
  - Count Plot
- Performed bivariate analysis
  - Scatter Plot
  - Grouped Box Plot
- Performed multivariate analysis
  - Correlation Heatmap
- Summarized key insights and model readiness

## Key Findings
- Most laptops are priced between ₹40,000 and ₹90,000.
- Laptop price generally increases with specification rating.
- HP, Lenovo, and Asus are the most common laptop brands in the dataset.
- The dataset contains no missing values or duplicate records.

## Project Files
- `EDA_Assignment.ipynb` – Jupyter Notebook containing the complete EDA.
- `data.csv` – Dataset used for analysis.
